# POSTGRES - SQL

Connect to db
```
$ sudo su postgres
$ psql
```

### Commonly used commands
`\c <name of database>` connect to database

`\l`  list all databases

`\l+` list all databases with additional information

`\d`  list tables

`\d+` describe table (list columns and datatypes)

`\du` list users

`\q`   quit
